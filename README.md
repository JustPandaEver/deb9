
### OS
* Debian 9

### Install
**For Update & Upgrade Binary**
```
apt-update && apt-upgrade
```
**Download / Install Git**
```
apt-get install git
```
**Cloning This Repo**
```
git clone https://github.com/JustPandaEver/deb9
```
**Open Folder "deb9" & move deb9.sh to root**
```
cd deb9 && mv deb9.sh /root or ..
```
**Execute Script**
```
chmod +x deb9.sh && ./deb9.sh
```
